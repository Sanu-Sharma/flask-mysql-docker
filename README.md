# Flask + MySQL Docker App 🐳

## About
Two-tier web application built with Flask and MySQL, 
containerized using Docker and deployed on AWS EC2.

## Tech Stack
- 🐍 Python Flask (Backend)
- 🐬 MySQL (Database)
- 🐳 Docker & Docker Networking
- ☁️ AWS EC2 (Deployment)

## How to Run
```bash
docker-compose up -d
```
